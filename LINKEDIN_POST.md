I'm excited to share my latest project: The AI-Powered Quantum Levitation Simulator! 🚀⚛️

I wanted to blend my passion for UI/UX design with complex physics simulations, so I built a standalone desktop application from the ground up using Python and the Flet framework. 

The simulator lets you act as a quantum systems engineer in a futuristic "Mission Control" interface (heavily inspired by SpaceX's deep space telemetry aesthetics). You can dynamically adjust magnetic fields and superconductor temperatures to achieve the Meissner effect (quantum levitation). 

But the coolest part? I integrated the Llama 3 70B AI model via the Groq API directly into the app. Based on the real-time physics parameters you set, the AI acts as your onboard science officer, providing rigorous, dynamic analysis on field stability and thermal management.

Tech Stack:
🔹 Python 
🔹 Flet (Flutter for Python) for the sleek, cross-platform GUI
🔹 Groq API (Llama 3 70B) for instant AI inference
🔹 PyInstaller for packaging it into a standalone Windows .exe
